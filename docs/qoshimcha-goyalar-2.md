# 🚀 ҚЎШИМЧА ҒОЯЛАР (раунд 2) — тизимни кейинги даражага

_Сана: 2026-06-26. Чуқур таҳлил: тизим ҳозир «норма ўрганади» → кейинги қадам «башорат + оптимизация»._

---

## 🔮 1. АҚЛЛИ БАШОРАТ (forecast) — ⭐ ФЛАГМАН ҒОЯ
Тизим аллақачон тарихдан норма ўрганади. **Кейинги қадам — БАШОРАТ:**
- Эртага қанча савдо бўлади? (кун + ҳафта куни + об-ҳаво + байрам)
- → **бозорчига:** қанча гўшт олиш керак
- → **шашликчига:** қанча сих тайёрлаш керак
- → **менежерга:** қанча смена/ходим

> **Фойда:** ортиқча тайёрлаб **бузилмайди**, кам қолиб **тугаб қолмайди**. «Аqлли тизим» ғоясининг чўққиси. Рустам ака эрталаб: «бугун ~22 млн кутилмоқда, 60 кг гўшт, 200 сих» деб кўради.

## 📊 2. ТАОМ ФОЙДА ТАҲЛИЛИ (menu engineering)
Ҳар таом: фойда % + сотув ҳажми → **4 гуруҳ:**
- ⭐ Юлдуз (фойдали + кўп) — **тарғиб қил**
- 🐴 От (кам фойда + кўп) — **нарх ошир** (Марварид, Кусковой)
- ❓ Жумбоқ (фойдали + кам) — **кўпроқ сот**
- 🐕 Ит (кам фойда + кам) — **олиб ташла**

> Бизда `moliyaviy-tahlil` бор — буни **жонли дашборд** қиламиз. Менюни пул ишлайдиган қилади.

## ⏰ 3. ПИК СОАТ / КУН ТАҲЛИЛИ
Қайси соат/кун гавжум → смена режа + олдиндан тайёргарлик. «Жума кечаси гавжум → кўпроқ сих, кўпроқ официант».

## 📱 4. TELEGRAM BOT (директорга)
- 🚩 Критик тешик → дарров Telegram'га
- 📊 Кун охири авто ҳисобот → Telegram (тушум/фойда/тешик/штраф)
- Рустам ака телефонни очмай, Telegram'дан кўради

## 🍽️ 5. QR-МЕНЮ + МИЖОЗ ЎЗИ ЗАКАЗ
Мижоз стол QR'идан менюни кўриб, **ўзи заказ беради** → камроқ официант, тезроқ, хато кам. *(Clopos'да бу модул бор эди — биз ишлатамиз.)*

## ⏲️ 6. КУХНЯ ТАЙЁРЛАШ ВАҚТИ + KDS
- Ҳар таом **норма вақти** (мас. шашлик 15 мин) → кечикса 🚩 (кухня секин)
- KDS экран (принтер ўрнига) — заказлар экранда, тайёр бўлгани белгиланади

## 📅 7. СМЕНА РЕЖА (график)
10 официант навбатини **пик соатга мослаб** планлаш. Ким қачон ишлайди — олдиндан.

## 📆 8. МУДДАТ НАЗОРАТИ (expiry)
Маринад/сут/гўшт **муддати** → бузилишдан олдин огоҳ (FIFO билан). «Маринад 4-кун — бугун сот».

## 🎂 9. МИЖОЗ CRM + ТУҒИЛГАН КУН
Доимий мижоз + телефон + туғилган кун + тарих → SMS таклиф. Қарз мижозлари билан уланади. Доимий мижозга бонус.

## 💹 10. ОЙЛИК МАҚСАД (target)
Директор ойлик фойда мақсади қўяди → реал-вақт прогресс («бу ой 165 млн мақсад, 120 млн бажарилди, 10 кун қолди»).
