### 🛒 БОЗОРЧИ роли — тўлиқ

**Ким:** бозордан харид қилиб келадиган одам (снабжение).

**Нима қилади:**
- Бозордан **гўшт** (қўй/мол) олади — ҳар сафар бошқа қассобдан бўлиши мумкин
- **Сабзавот, кўкат, картошка, пиёз** олади
- **Кўмир** (мангал учун), **лаваш/нон**, **зиравор** олади
- Ҳар харидни **дастурга киритади**: нима, неча кг/дона, нархи, етказувчи (ихтиёрий — «бозор» + изоҳ), тўлов тури (нақд/қарз)
- Ёмон маҳсулотни **қайтаради** ёки чегирма билан олади

**Тизимда нима кўради:**
- Фақат **харид/кирим** модули. Таннарх, сотув, фойдани **КЎРМАЙДИ**.

**Харид қоидалари (раунд 3 да тасдиқланган):**
- **Етказувчи** — мажбурий эмас (ҳар сафар бошқа қассоб; «бозор» + изоҳ етади)
- **Нарх** — ҳар туша ўз нархи билан (битта кунда 2 туша ҳар хил нархда бўлиши мумкин; ўртача эмас)
- **Тўлов** — нақд (дарров) ёки **қассобга қарз** (кредиторлик — ихтиёрий, «тўланди/қарз» белгиси)
- **Кўмир/газ** — таомга эмас, **умумий харажат** (OPEX)

**Оқим:** Бозорчи бозорга боради → маҳсулот олади → дастурга кирим қилади → **омборга тушади**. Гўшт бўлса → 2 повар обвалка → менежер вазн ёзади → таннарх чиқади.

### 📍 Харид локацияси (геолокация) — назорат

**Ғоя:** Бозорчи телефонда харидни киритганда, дастур **қаерда ва қачон** олганини авто ёзади. Директор «қаердан, неча пулга олди» ни аниқ кўради — алдаб бўлмайди.

**Авто ёзилади (бозорчи телефонидан):**
- 📍 **GPS локация** — қайси бозор/жой (харитада нуқта)
- 🕐 **Вақт** — қачон киритди (реал-вақт; кейин ўзгартириб бўлмайди)

**Ихтиёрий (бозорчи бириктиради):**
- 📷 **Расм** — гўшт / чек / тарози экрани (исбот)
- ✍️ Бозор/қассоб номи
- 💬 **Изоҳ (ихтиёрий)** — нарса олгандан кейин эслатма: масалан «гўшт сифати яхши», «нарх ошган», «доимий қассоб йўқ эди, бошқадан олдим»

**Директор кўради:** харитада бозорчи қаерга борди, ҳар жойдан нима/неча пулга олди, қачон, расм билан.
- 📷 Расм = **ихтиёрий** (мажбурий эмас)
- 🗺️ **Бозор харитаси (ойлик):** бозорчи ой давомида қаерларга борди, қаердан қанча сарфлади — директорга харита+ҳисобот

### 🛡️ Бозорчи назорат функциялари (5 та — ҳаммаси тасдиқланди)
1. 🔴 **Нарх аномалияси:** дастур ҳар маҳсулот нархини тарихдан ўрганади → қиммат киритилса «Қўй +12% қиммат (165к vs одатда 147к)» 🚩
2. 📝 **Авто харид рўйхати:** омбор қолдиғи камайса → «олиш керак» рўйхати ўзи тузилади (бозорчига)
3. ⚖️ **Харид↔обвалка солиштируви:** «35кг олди, обвалкада 33кг чиқди → 2кг фарқ» 🚩 (йўлда йўқолиш/кам тортиш)
4. 💰 **Кунлик бюджет:** директор лимит беради (мас. 6млн); ошса → огоҳлантириш
5. ⭐ **Қассоб рейтинги:** нарх+сифат+қайтаришлардан рейтинг (🟢🟡🔴) — ким яхши/арзон беради

**Назорат фойдаси:** GPS'ни яшириб, вақтни сохталаштириб бўлмайди. «Қиммат бозордан олдим» деса — харита текширади.

> ⚠️ **Нозиклик:** локация аниқ бўлиши учун бозорчи **бозорда турганда** (реал-вақт) киритиши керак. Ресторанга келиб киритса — GPS ресторанни кўрсатади (бу ҳам шаффоф кўринади). Дастур бозорчи телефонидан **локация рухсати** сўрайди.

## 1️⃣ WORKFLOW: ГЎШТ ХАРИД + ОБВАЛКА

```
БОЗОРЧИ бозордан харид қилади (гўшт, сабзавот, кўмир, лаваш, зиравор)
        │
        ▼
БОЗОРЧИ харидни КИРИТАДИ:  тур + кг/дона + нарх + етказувчи(ихтиёрий) + тўлов(нақд/қарз)
   (ҳар туша ўз нархи билан; қассоб ҳар сафар бошқа бўлиши мумкин)
        │
        ▼
2 ПОВАР обвалка қилади (туша → қисмлар), МЕНЕЖЕР вазнни ёзади
        │
        ▼
ЭЛЕКТРОН ТАРОЗИ ҳар қисмни тортади → вазн АВТО дастурга тушади
   ├─ Премиум кесим: Вағора, Шашлик, Корейка (учаласи алоҳида, бошқа нарх) — СОТИЛАДИ
   ├─ Шапок / Базон-к / Буғлама — СОТИЛАДИ
   ├─ Думба (фақат қўйда), Фарш, Жигар — СОТИЛАДИ
   ├─ Суяк → мангалга, СОТИЛАДИ (алоҳида нархи кейин киритилади; ҳозирча cost=0)
   ├─ Чарви → ФАРШга ишлатилади (ҳар доим эмас, ad-hoc); cost=0
   └─ Мусор/брак → яроқсиз, ҳисобдан чиқади, cost=0
        │
        ▼
ВАЗН БАЛАНСИ ёпилади:  Σ(қисмлар) ≈ туша вазни  (фарқ >±5% → 🚩 флаг)
        │
        ▼
АНОМАЛИЯ ТЕКШИРУВИ: ҳар қисм чиқими тарихий норма билан солиштирилади
   (масалан "Маруф +13.9% фарш" → дастур ўзи ушлайди)
        │
        ▼
ОМБОРГА КИРИМ: барча қисмлар муздаткичга, ҳар тушанинг ўз нархи билан
```

**Таннарх ҳисоби (тўлиқ модель):**
- Туша нархи **сотиладиган гўшт қисмлари**га тарқалади (премиум, шапок, думба, фарш, жигар...).
- **Суяк, чарви, брак — cost = 0** → уларнинг вазн-нархи гўштга юкланади (премиум тўғри қимматлашади).
- **Суяк сотилганда** (нархи кейин) → таннархи ~0 бўлгани учун **деярли соф фойда (бонус)**.
- **Чарви** фаршга кетганда → фаршга деярли текин кириш (cost≈0).
- Йўқотиш % ҳисобга олинади → ҳар қисмнинг **реал 1 кг таннархи** чиқади.

## 2️⃣ WORKFLOW: МАРИНОВКА / ИШЛАБ ЧИҚАРИШ (ярим-тайёр)

```
ХОМ ГЎШТ (омбордан)  ──►  МАРИНОВКА (партия)  ──►  СИХЛАШ
        │                                              │
   омбордан ЧИҚАДИ                          «маринадланган шашлик»
   (ишлаб чиқариш ҳужжати)                  = ЯРИМ-ТАЙЁР SKU, омборда туради
```

**Ярим-тайёр маҳсулотлар (алоҳида қолдиқ) — ШАШЛИКЧИ тайёрлайди:**
- **Маринадланган/сихланган шашлик** — хом лахм → мариновка → сих
- **Фарш** — ad-hoc (ҳар сафар бошқача рецепт): Шапок 50–100кг + ~10% қорин чарви + пиёз (1.7% ёз / 2.0% қиш)
- **Рулет** (1.6 кг/дона): думба 150г + гўшт 550г + фарш 900г
- **Вафли** (1 паднос): лаваш 6×120г + фарш 6.1кг → порцияга бўлинади (~140г/порция)

> 🔪 **Ким қилади:** Фарш, Рулет, Вафли + мариновка/сихлаш = **ШАШЛИКЧИ**. Обвалкани эса 2 повар қилади (менежер вазн ёзади).

> ⏳ **Мариновка партияси ўзгарувчан (савдога қараб):** бир сафар **2–4 кунлик** тайёрланади (савдо зўр бўлса 2 кунга, паст бўлса 3–4 кунга). Демак маринадланган шашлик омборда бир неча кун туради → дастур **FIFO** (аввал тайёрланган аввал сотилади) + 🚩 эски маринад огоҳлантириши (**неча кунлиги — кейин созланади**).

> **Муҳим қоида:** ишлаб чиқаришда хом-ашё омбордан чиқади, тайёр ярим-тайёр кирим бўлади. Сотув-списание эса кейинроқ — чек ёпилганда.

### 🍢 СИХ / ВИТРИНА — уч босқичли қолдиқ

Шашликчи **олдиндан** теради → **витринага** қўяди, қолгани **маринадда** туради. Демак сих учун **3 босқич қолдиқ:**
```
1. ХОМ ГЎШТ (обвалкадан)      — кг
       ↓ мариновка (ишлаб чиқариш)
2. МАРИНАДДА (маринадланган)   — кг
       ↓ сихлаш (ишлаб чиқариш)
3. ВИТРИНАДА (сихланган, тайёр) — ДОНА
       ↓ сотув (чек ёпилганда)
   СПИСАНИЕ — витринадан (дона)
```
Ҳар ўтиш = ишлаб чиқариш ҳужжати (менежер ёзади). Фарш/рулет/вафли ҳам **заготовка** билан → тайёрлаб → витринага.

### 🛡️ Витрина назорат вариантлари (танлаш керак)
1. ⚖️ **Витрина ↔ сотув солиштируви (энг кучли):** кун охири — сихланган (ишлаб чиқарилган) − сотилган (чек) = витринада қолиши керак. Мос келмаса → 🚩 «сих йўқолди / текин кетди / нотўғри саноқ». **Сих пулсиз чиқиб кетишини ушлайди.**
2. ✍️ **Витрина саноғи қўлда:** менежер витринадаги сих сонини **қўлда** киритади (фото шарт эмас).
3. 🚦 **Витрина светофори:** машҳур сих кам қолса → 🟡/🔴 шашликчига «яна тайёрла» (омбор светофоридек).
4. 📊 **Сих аналитикаси:** қайси сих кўп сотилади, пик соатлар → ортиқча тайёрлаб бузилмайди.
5. ⏳ **Витрина FIFO + муддат:** аввал терилган аввал сотилади; узоқ турса 🚩 эски сих.

### 🧮 «Ҳисобланган сих» ↔ «Қўлда киритилган сих» — АДАШМАЙДИ (бир-бирини текширади)

Сенинг саволинг: дастур 10 кг кусковойдан граммга бўлиб сих сонини чиқаради, биз ҳам сихни киритсак — адашмайдими? **Йўқ.** Иккиси **ҳар хил вазифа** учун:

- **Ҳисобланган (норма):** 10 000 г ÷ 100 г/сих = **100 сих кутилади** (норма)
- **Қўлда киритилган (реал):** шашликчи теради, менежер **реал сонни** киритади (мас. 97 сих)
- **Дастур СОЛИШТИРАДИ:** кутилган 100 ↔ реал 97 = −3 (3%). Допуск ичида → ✅. Кўп бўлса → 🚩

**Қолдиқ адашмайди, чунки иккита ҳар хил бирлик:**
- Маринад = **кг** (10 кг −) · Витрина = **дона** (97 сих +). Гўшт икки марта саналмайди — сихлаш ҳужжати уларни боғлайди.
- **Реал сон** қолдиқни юритади (сотувга шу кетади). **Ҳисобланган сон** фақат «текширув/норма» — гўшт камаймадими, граммни оширмадими.

> **Мисол:** 10 кг дан 100 сих кутилади. Менежер 85 ёзса → 🚩 «117 г/сих ёки 1.5 кг гўшт камайди — қаёққа кетди?». Шу билан **сих грамми ва йўқолиш** ушланади (обвалка ±5% дек).

## 3️⃣ WORKFLOW: СОТУВ (зал ва собой)

```
ОФИЦИАНТ телефонда буюртма олади
   (стол/зал танланади → хизмат ҳақи ҳар зал бўйича:
    Асосий зал 10% · Катта зал 10% · Терраса 15% · Собой 0%)
        │
        ▼
КУХНЯ ПРИНТЕРИ — мангалга/ошхонага чоп этилади  ◄── (ҳали списание ЙЎҚ)
        │
        ▼
ТАЙЁРЛАНАДИ → мижозга берилади
        │
        ▼
КАССИР «Ёпиш» босади → ТЎЛОВ ЭКРАНИ → тур ТАНЛАШ (МАЖБУРИЙ):
   [Нақд] [Пластик] [Хумо] [Клик] [Қарз]  — кўп-тўлов (мас. 200к нақд + 100к қарз)
   ⛔ тур танланмаса чек ЁПИЛМАЙДИ · қарз → мижоз мажбурий
        │
        ▼
✅ ЧЕК ЁПИЛАДИ  ──►  🔻 СПИСАНИЕ ШУ ЕРДА:
   тех-карта бўйича ярим-тайёр/ингредиент омбордан АВТО ечилади
   + хизмат ҳақи (зал 10%) авто қўшилади
        │
        ▼
АТРИБУЦИЯ: официант (+1 буюртма) ва кассир (+сумма) — АЛОҲИДА ёзилади
```

**Муҳим:**
- Бекор қилинган / тўланмаган буюртма → омборга **тегмайди** (списание йўқ).
- Собой ҳам официант + кассир орқали ўтади (хизмат ҳақи 0%).
- Ўчирилган таом → журналга, директорга рўйхат.

### 📱 Официант мобил POS — батафсил (ядро функция)

Официант **ўз телефонида** POS'дан фойдаланиб заказ олади:
```
1. PIN билан киради (шахсий) → буюртма ўзига боғланади
2. Стол танлайди (зал/катта зал/собой) — зал тури хизмат ҳақини белгилайди
3. Менюдан таом танлайди (категория: таом/шашлик/салат/ичимлик...)
4. Сони + изоҳ ёзади (мас. "аччиқ эмас", "тузсиз", "яхши пишсин")
5. ЮБОРАДИ → ҳар таом ўз станция принтерига чоп (мангал/ошхона/салат/бар)
6. Кейин таом ҚЎШИШ мумкин (қўшимча буюртма) — эркин
7. Таом ОЛИБ ТАШЛАШ (чопдан кейин) → ДИРЕКТОР рухсати (журналга)
```
- Официант **пулга тегмайди** — тўловни кассир олади
- Битта официант бир неча столга хизмат қила олади
- Заказ → атрибуция: ким олди (официант) кўринади
- Оффлайн ишлайди (интернет ўчса ҳам заказ олинади, улангач синхрон)

## 4️⃣ WORKFLOW: ҚАРЗ ДАФТАРИ (подробно)

### Асосий ғоя
Ҳар бир мижознинг **битта умумий қолдиғи** бор (алоҳида чеклар рўйхати эмас). Қарз = **манфий қолдиқ**. Ҳар тўлов қолдиқни нолга яқинлаштиради. Лекин ҳар бир ҳаракат (қарз олиш + тўлов) **ўзгармас ёзув** бўлиб журналда қолади — тарихни кўриш учун.

### Қадам-бақадам

**1-қадам — Қарз очилади (чек ёпилганда):**
```
Кассир чекни ёпаётганда тўлов турини танлайди.
"Қарз" танланса → МАЖБУРИЙ: мижоз бириктирилади (исм + телефон).
Аралаш ҳам бўлади: 300к чек = 200к нақд + 100к қарз
   → 100к ўша мижоз қолдиғига -100к бўлиб ёзилади.
```

> 🧾 **Чек қарзга бириктирилади:** ҳар қарз ёзуви ўз ЧЕКИ билан сақланади — **нима олган, қанча, қачон, қайси официант/кассир**. Кейин "бу қарз нимадан чиққан?" деган савол қолмайди — чек ўша ерда исбот бўлиб туради. Директор қарздорни очса, ҳар қарзнинг асл чекини кўра олади.

> 💬 **Изоҳ (комментария) майдони:** кассир/официант ҳар қарзга (ёки чекка) эркин эслатма ёзиб қўяди — масалан *«Акмал жўраси билан келди»*, *«оиласи билан келди»*, *«туғилган кун»*. Бу вазиятни кўрсатади, кейинроқ директор контекстни тушунади. Изоҳ ҳам чек+қарз билан бирга сақланади.

**2-қадам — Гўшт барибир ечилади ⚠️:**
```
Қарз = ФАҚАТ ПУЛ масаласи. Овқат ейилди → списание чек ёпилганда барибир бўлади.
Яъни қарзга сотилган таом ҳам омбордан ечилади (тех-карта бўйича).
Қарз қолдиқ/омборга таъсир қилмайди — фақат кассага.
```

**3-қадам — Қисман тўлов:**
```
Мижоз 500к қарздор. 200к олиб келди → қолдиқ -300к бўлади.
Қайси чекка деб ажратиш ШАРТ ЭМАС — умумий қолдиқ ҳаракатланади.
Лимит ЙЎҚ — қанча қарз берса шунча ёзилаверади.
```

**4-қадам — Қарз ёпилади:**
```
Қолдиқ 0 га етганда қарз ёпилди. Кассир тўловни киритади.
Тўлов тури ҳам ёзилади (нақд/Click/Payme билан тўладими).
```

### Атрибуция (муҳим назорат)
Ҳар қарзда **қайси кассир берган** ёзилади → директор "Дилноза кўп қарз беряпти" ёки "фалон мижозга жуда кўп қарз йиғилган" деб кўради.

### Директор нима кўради
| Кўрсаткич | Изоҳ |
|---|---|
| Жами қарз | Ҳамма мижозлардан йиғилган умумий қарз |
| Ким қанча | Ҳар мижоз бўйича қолдиқ |
| Қачондан бери | Эскилиги (aging) — 1 ой+ тўламаганлар қизил |
| Ким берган | Қайси кассир очган |
| Тарих | Ҳар олиш ва тўлов — тўлиқ журнал |

### Назорат қоидалари
- Қарз қолдиғи **ўзи ҳисобланади** (журналдан) — қўлда ўзгартириб, яшириб бўлмайди.
- Қарзга сотув **ҳақиқий фойда**га киради, лекин **кассадаги пул**га кирмайди → директор иккаласини алоҳида кўради (кассада пул кам, лекин катта қарз бор = хавфли).

> **Мисол:** Мижоз "Акмал" 3 марта келди: 150к + 200к + 100к қарз = қолдиқ **-450к**. Кейин 300к тўлади → **-150к**. Директор: Акмал 150к қарздор, биринчи қарз 12 кун олдин, ҳаммасини кассир Дилноза берган.

## 5️⃣ WORKFLOW: КУН ЁПИЛИШИ (06:00)

```
Смена БИТТА (ҳозирча битта кассир; кейин кўпайтириш мумкин)
Бошида касса размени = ~50 000 (сдача учун)
        │
        ▼ КУН ДАВОМИДА:
КАССАДАН ЧИҚИМ — директор пул олади (кўп нарсага) → ҲАР БИРИ ЁЗИЛАДИ:
   тур танланади:
   🛒 Харид тўлови (етказувчига) — фақат пул ҳаракати (харид аллақачон кирим, харажат эмас)
   💸 Харажат (ремонт, идиш, газ...) — P&L'га харажат
   👑 Эга олди (фойда) — эга ҳисобига, харажат эмас
   + сумма + изоҳ + ким олди
        │
        ▼ 06:00:
КАССИР кунни ёпади → нақд санайди → СЕЙФга
        │
        ▼
КУТИЛГАН НАҚД = размен + нал сотув − кассадан чиқимлар − возврат
   реал саналган ↔ кутилган → ФАРҚ?
        │
        ▼
КАМОМАД → ДИРЕКТОРГА (қарор директорники). Кассир санайди → ДИРЕКТОР тасдиқлайди.
```

> 💸 **Муҳим:** кассадан чиқим **тур билан** ёзилгани учун — харид тўлови иккиланмайди, харажат P&L'га тушади, эга олгани фойдадан ажралади. Шу билан камомад **тўғри** чиқади.

> ⚠️ **Эслатма:** кун ёпилиши (06:00) = **касса** (кассир). **Инвентаризация алоҳида — ЭРТАЛАБ**, менежер қилади (қуйида).

### 📦 КУНЛИК ИНВЕНТАРИЗАЦИЯ (Менежер, ЭРТАЛАБ)
```
Эрталаб (очилишда) → менежер ҲАММА маҳсулотни санайди
   дастур ҳисоблаган қолдиқ ↔ реал санаб киритилган
   фарқ → сабаб ёзилади → ДИРЕКТОР тасдиқлайди
        ↓
   натижа → харид рўйхати (нима кам) → бозорчига
```

### 🚦 ОМБОР СВЕТОФОРИ (менежер тизимида)
Ҳар маҳсулот **ранг** билан кўринади — бир қарашда нима кам:
- 🟢 **Яшил** — етарли (3+ кунга етади)
- 🟡 **Сариқ** — камаймоқда (1–2 кунга, эрта олиш керак)
- 🔴 **Қизил** — кам/тугади (бугунга етмайди — ҲОЗИР олиш керак)

**Ранг қандай аниқланади (аqлли):** дастур ҳар маҳсулотнинг **кунлик ўртача сарфини тарихдан** ўрганади → қолдиқ ÷ кунлик сарф = «неча кунга етади» → ранг.
> Мисол: Гўшт қолдиқ 8кг, кунлик 56кг → 🔴 · Помидор 15кг, кунлик 5кг → 🟢 · Пиёз 4кг, кунлик 3кг → 🟡

**Уланиш:** 🔴 ва 🟡 маҳсулотлар **авто «олиш керак» рўйхатига** тушади (бозорчига). **Чегара:** авто (тарихдан) + директор ҳар маҳсулотга қўлда мин даража қўя олади.
