## 🥩 ОБВАЛКА БАЙПРОДУКТ — таннарх модели ТЎЛИҚ ёпилди

| Қисм | Қарор | Таннарх |
|---|---|---|
| **Суяк** | Мангалга → **сотилади**. Алоҳида нархи бўлади, **кейин киритилади** (келажак план, ҳозир нарх аниқ эмас) | cost = 0 (нархи гўштга), сотуви ~соф фойда |
| **Чарви** | **Фаршга** ишлатилади, лекин **ҳар доим эмас** (ad-hoc) | cost = 0 (фаршга деярли текин кириш) |
| **Брак / мусор** | Яроқсиз — шунчаки **ҳисобдан чиқади** | cost = 0 |

**Хулоса:** туша нархи фақат **сотиладиган гўшт қисмлари**га тарқалади (премиум, шапок, думба, фарш, жигар). Суяк/чарви/брак нархи 0 → премиум гўшт тўғри қимматлашади. Суяк сотилганда (нархи кейин) — бонус фойда.

## 💰 КУХНЯ САВДОСИ 30/70 МОДЕЛИ (стратегик таҳлил)

Рустам ака савол: «Кухня савдосини 30% менга / 70% поварга қилсам нима бўлади? Статистикадан кўрсак».

|---|---:|---:|

### 30/70 бўлиниш (фақат иссиқ кухня):
| | Ойига | Кунига |
|---|---:|---:|
| 👨‍🍳 Повар 70% | 232.5 млн | ~7.75 млн |
| 👑 Рустам ака 30% | 99.6 млн | ~3.32 млн |

### 🔴 Ҳал қилувчи савол: ким гўшт пулини тўлайди?
- **А) Повар ўз 70%идан гўшт олади** (ошпаз аренда): Рустам ака 30% = тоза фойда + ичимлик/спиртли 100% (~107 млн/ой) → жами **~207 млн/ой кам рискли**. Поварга гўштдан кейин ~15% қолади.
- **Б) Гўштни Рустам ака олади:** 30% (99.6) − гўшт (~182) = **МИНУС** ❌ — тўғри келмайди.

### Очиқ (аниқлаш керак):
1. «Кухня савдоси»га нима киради — фақат таом+шашликми, ёки салат/балиқ ҳам?
2. Бу моделда гўштни ким олади — повар ёки Рустам ака?

> Дастур қурилгач: Рустам ака бундай вариантларни (30/70, 40/60, категория қўш/олиб таш) **ўзи ўйнаб**, дарров натижани кўради. Бу — "what-if" аналитика кучи.
