### 🏗️ Milestone 0 — Пойдевор
- Next.js + Supabase лойиҳа
- **Auth: PIN-логин** (хэшланган, кучли) + рол-асосли рухсат (13 фойдаланувчи)
- **Кирилл ўзбекча** UI каркаси
- **Offline-first** асос (локал кэш + синхрон механизми)

### 📋 Milestone 1 — Каталог + Рецепт
- Supabase схема: маҳсулот (INGREDIENT/PART/SEMI/DISH/GOODS), категория, станция, локация, ҳаракат
- **Clopos каталог импорт:** 235 таом + категория + станция + нарх (тозалаб: 9 ўчир + дубликат + ШАШЛИКЛАР→14)
- **Рецепт импорт:** `texkarta.json` → "Состав" (ингредиент + грамм) — Clopos'да бу бўш эди!

### 🔪 Milestone 2 — ОБВАЛКА + ТАННАРХ ⭐ (ЯДРО)
- Туша → қисмлар (**1-босқич қўлда вазн**, кейин тарози интеграцияси)
- **±5% вазн баланси** + аномалия (`obvalka-history.json` дан норма)
- **Таннарх движок:** туша нархи → сотиладиган гўшт қисмларига (суяк/чарви/брак cost=0) → реал 1кг таннарх
- Мариновка ўсиши (+13–30%) + йўқотиш % ҳисоби

### 🍢 Milestone 3 — Сих / Витрина
- **3 босқичли қолдиқ:** хом гўшт(кг) → маринад(кг) → витрина(дона)
- Ишлаб чиқариш ҳужжатлари (мариновка, сихлаш, фарш/рулет/вафли)
- **«Ҳисобланган ↔ қўлда» сих солиштируви** (грамм назорати)
- **Омбор светофори** 🟢🟡🔴 (авто, тарихдан)

### 💳 Milestone 4 — POS (сотув)
- **Официант мобил:** PIN → стол(зал) → меню → заказ → станция чоп · offline
- **Кассир:** кўп-тўлов (нақд/Click/Payme/қарз) → **чек ёпиш = списание** + хизмат ҳақи (Асосий10/Катта10/Терраса15/Собой0)
- **Атрибуция** (официант+кассир алоҳида)
- **Мажбурий:** тикетсиз таом ЙЎҚ

### 💰 Milestone 5 — Молия + Касса + Қарз (Excel'дан кенгайди)
- **Кунлик касса журнали** (Excel'дек, Вариант C): кун боши қолдиқ → кирим(POS авто) → чиқим(категория) → кун охири → камомад
- **Харажат категориялари:** 💵кунлик иш ҳақи(алоҳида экран) · 🛒бозорлик · 💧коммунал(газ/сув/кўмир) · 👑эга олди · 📦бошқа
- ⭐ **P&L ажратиш:** бозорлик→COGS · иш ҳақи/коммунал→OPEX · эга олди→тақсимот → **реал фойда**
- **Бир амал икки ёзув:** бозорчи харид → касса − + омбор +
- **Қарз ledger** (чек+изоҳ+SMS+рейтинг) + Excel'дан бошланғич қарздорлар импорт
- OPEX (126млн/ой) + кун ёпилиши (06:00)

### 🍽️ Milestone 7 — Посуда инвентаризацияси (1.5-босқич)
Идиш-товоқ саноғи (~2180 дона) + топшириқ-қабул + синган/йўқолган назорати (янги тешик).

### 🛡️ Milestone 6 — Назорат + Директор панели
- **Кунлик инвентаризация** (эрталаб, менежер)
- **Вазифа назорати + штраф** (3 тур, жонли камера)
- **Тешиклар движок** (23 та) + критик → push
- **Директор панели:** 4 рақам · остаток(светофор) · тешиклар · реал фойда · тасдиқлар
