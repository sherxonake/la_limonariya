## 🎯 ЛОЙИҲА ҚИСҚАЧА

**Муаммо:** Clopos POS таннарх/омборни билмайди → «фойда» 97% сохта (реалда ~27%), склад бузуқ (−232М).

**Ечим:** тўлиқ янги POS — касса+меню+тўлов+чек **ПЛЮС** кирим→обвалка→**реал таннарх→реал қолдиқ→реал фойда**→дастур ўзи **тешик/аномалия топади + башорат қилади**.

**Технология:** Next.js + Supabase · offline-first (PWA) · кирилл ўзбекча · 3 десктоп + 10 телефон.

**Калит:** энг катта пул рычаги — **шашлик грамми** (назорат ~25-28 млн/ой). Фискал: таом=ички, ичимлик=расмий (маркировка).
